# Bootstrap

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/laurensent/bootstrap/main/install.sh | bash
curl -fsSL "https://raw.githubusercontent.com/laurensent/bootstrap/main/install.sh?$(date +%s)" | bash  # 跳过缓存
```

## Uninstall

```bash
curl -fsSL https://raw.githubusercontent.com/laurensent/bootstrap/main/uninstall.sh | bash
```
